# Calculadora Python

Calculadora simples para uso local, feita em Python com interface gráfica usando Tkinter.

## Funcionalidades

- Soma, subtração, multiplicação e divisão
- Botão para limpar o campo
- Suporte a teclado:
  - números e operadores `+`, `-`, `*`, `/`
  - ponto ou vírgula para números decimais
  - parênteses para organizar cálculos
  - `Enter` calcula o resultado
  - `Backspace` apaga o último caractere
  - `Esc` limpa o campo
- Não precisa instalar bibliotecas externas

## Requisitos

- Python 3 instalado
- Tkinter disponível na instalação do Python

No Windows, o Tkinter normalmente já vem instalado junto com o Python.

## Como executar

Abra o terminal na pasta do projeto e rode:

```bash
python calculadora.py
```

Se o comando acima não funcionar, tente:

```bash
py calculadora.py
```
